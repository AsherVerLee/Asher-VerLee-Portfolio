Code Library

HTML & CSS Reference Guide for Intro to CS

Document Structure
<!DOCTYPE html>
Declares the document type and HTML version. Must be the first line of your HTML file.
<!DOCTYPE html>
<html>
Root element that contains all other HTML elements on the page.
<html lang="en">
  <!-- All content goes here -->
</html>
<head>
Contains metadata about the document, like title, character set, and links to stylesheets.
<head>
  <title>My Page</title>
  <meta charset="UTF-8">
</head>
<body>
Contains all visible content on the webpage (text, images, links, etc.).
<body>
  <h1>Welcome!</h1>
  <p>This is my page.</p>
</body>
<title>
Defines the title shown in the browser tab. Goes inside the <head> section.
<title>My Portfolio Website</title>
<meta>
Provides metadata about the HTML document (character set, viewport settings, description).
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="My portfolio">
Text & Headings
<h1> - <h6>
Heading tags from largest (h1) to smallest (h6). Used to structure content hierarchy.
<h1>Main Title</h1>
<h2>Subtitle</h2>
<h3>Section Heading</h3>
<p>
Defines a paragraph of text. Automatically adds spacing before and after.
<p>This is a paragraph of text.</p>
<strong>
Makes text bold and indicates strong importance.
<p>This is <strong>very important</strong> text.</p>
<em>
Emphasizes text with italics.
<p>This is <em>emphasized</em> text.</p>
<br>
Creates a line break. Self-closing tag (no closing tag needed).
First line<br>Second line
<hr>
Creates a horizontal rule (line) to separate content. Self-closing tag.
Section 1<hr>Section 2
Links & Navigation
<a>
Creates a hyperlink to another page or location. Uses href attribute for destination.
<a href="https://example.com">Visit Example</a>
<a href="page2.html">Go to Page 2</a>
<a href="#section">Jump to Section</a>
<nav>
Defines a navigation section containing links to other pages or sections.
<nav>
  <a href="home.html">Home</a>
  <a href="about.html">About</a>
</nav>
Images & Media
<img>
Embeds an image. Requires src (source) and alt (description) attributes. Self-closing.
<img src="photo.jpg" alt="Description of image">
<img src="images/logo.png" alt="Company logo" width="200">
<video>
Embeds a video player. Can include controls, autoplay, and loop attributes.
<video width="640" height="360" controls>
  <source src="video.mp4" type="video/mp4">
</video>
<audio>
Embeds an audio player.
<audio controls>
  <source src="song.mp3" type="audio/mpeg">
</audio>
<iframe>
Embeds another webpage or content (like YouTube videos) within your page.
<iframe src="https://example.com" width="600" height="400"></iframe>
Lists
<ul>
Creates an unordered (bulleted) list.
<ul>
  <li>First item</li>
  <li>Second item</li>
</ul>
<ol>
Creates an ordered (numbered) list.
<ol>
  <li>Step 1</li>
  <li>Step 2</li>
</ol>
<li>
Defines a list item within <ul> or <ol>.
<ul>
  <li>Item</li>
</ul>
Containers & Layout
<div>
Generic container for grouping content. Used with CSS for layout and styling.
<div class="container">
  <h2>Section Title</h2>
  <p>Content here</p>
</div>
<span>
Inline container for styling small portions of text or content.
<p>This is <span class="highlight">highlighted</span> text.</p>
<header>
Defines the header section of a page or section, typically containing navigation or titles.
<header>
  <h1>My Website</h1>
  <nav>...</nav>
</header>
<main>
Contains the main content of the webpage. Should only be used once per page.
<main>
  <h2>Main Content</h2>
  <p>Page content here</p>
</main>
<footer>
Defines the footer section, typically containing copyright, contact info, or links.
<footer>
  <p>© 2025 My Website</p>
</footer>
<section>
Defines a thematic grouping of content, typically with its own heading.
<section>
  <h2>About Me</h2>
  <p>Information...</p>
</section>
Forms & Input
<form>
Creates a form for user input. Contains input fields, buttons, etc.
<form action="/submit" method="POST">
  <input type="text" name="username">
  <button type="submit">Submit</button>
</form>
<input>
Creates an input field. Type attribute determines the kind of input (text, email, password, etc.).
<input type="text" placeholder="Enter name">
<input type="email" required>
<input type="password">
<button>
Creates a clickable button.
<button type="button">Click Me</button>
<button type="submit">Submit Form</button>
<label>
Defines a label for an input element. Improves accessibility.
<label for="username">Username:</label>
<input type="text" id="username">
Tables
<table>
Creates a table to display data in rows and columns.
<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
</table>
<tr>
Defines a table row.
<tr>
  <td>Cell 1</td>
  <td>Cell 2</td>
</tr>
<th>
Defines a table header cell (bold and centered by default).
<tr>
  <th>Header 1</th>
  <th>Header 2</th>
</tr>
<td>
Defines a table data cell.
<tr>
  <td>Data 1</td>
  <td>Data 2</td>
</tr>
Styling & Scripts
<link>
Links external resources like CSS stylesheets. Goes in <head> section.
<link rel="stylesheet" href="style.css">
<link rel="icon" href="favicon.ico">
<link rel="preconnect" href="https://fonts.googleapis.com">
<style>
Contains internal CSS styles within the HTML document.
<style>
  body { background-color: blue; }
  h1 { color: white; }
</style>
<script>
Embeds or links JavaScript code. Can go in <head> or end of <body>.
<script src="script.js"></script>
<script>
  console.log("Hello!");
</script>
Special Characters
HTML Entities
Special codes for characters that have special meaning in HTML or aren't on the keyboard.
&lt; = < (less than)
&gt; = > (greater than)
&amp; = & (ampersand)
&nbsp; = non-breaking space
&copy; = ¬© (copyright)
&hearts; = ‚ô• (heart)
&#8594; = ‚Üí (arrow)
Comments
<!-- comment -->
Creates a comment that won't be displayed on the page. Useful for notes and documentation.
<!-- This is a comment -->
<!-- TODO: Add more content here -->
<!-- Temporary: Will remove later -->
Text Properties
color
Sets the text color. Can use color names, hex codes, or RGB values.
h1 { color: blue; }
p { color: #FF0000; }
span { color: rgb(0, 255, 0); }
font-family
Specifies the font for text. Provide fallback fonts in case the first isn't available.
body { font-family: Arial, sans-serif; }
h1 { font-family: 'Times New Roman', serif; }
font-size
Sets the size of text. Can use px, em, rem, or %.
h1 { font-size: 32px; }
p { font-size: 1.2em; }
small { font-size: 80%; }
font-weight
Controls the boldness of text. Values: normal, bold, or numbers (100-900).
h1 { font-weight: bold; }
p { font-weight: 400; }
strong { font-weight: 700; }
font-style
Makes text italic or normal.
em { font-style: italic; }
p { font-style: normal; }
text-align
Aligns text horizontally: left, right, center, or justify.
h1 { text-align: center; }
p { text-align: left; }
.footer { text-align: right; }
text-decoration
Adds decorations to text: underline, line-through, or none.
a { text-decoration: none; }
.strike { text-decoration: line-through; }
.underline { text-decoration: underline; }
text-transform
Controls capitalization: uppercase, lowercase, or capitalize.
h1 { text-transform: uppercase; }
.title { text-transform: capitalize; }
line-height
Sets the spacing between lines of text.
p { line-height: 1.6; }
h1 { line-height: 1.2; }
letter-spacing
Adjusts spacing between characters.
h1 { letter-spacing: 2px; }
.spaced { letter-spacing: 0.1em; }
Colors & Backgrounds
background-color
Sets the background color of an element.
body { background-color: #f0f0f0; }
.box { background-color: lightblue; }
header { background-color: rgb(20, 33, 61); }
background-image
Sets an image as the background.
body { background-image: url('background.jpg'); }
.hero { background-image: url('images/banner.png'); }
background-size
Controls the size of background image: cover, contain, or specific dimensions.
.hero { background-size: cover; }
.banner { background-size: contain; }
.box { background-size: 100px 200px; }
background-position
Positions the background image.
.hero { background-position: center; }
.banner { background-position: top right; }
.box { background-position: 50% 20%; }
background-repeat
Controls if/how background image repeats: repeat, no-repeat, repeat-x, repeat-y.
.pattern { background-repeat: repeat; }
.banner { background-repeat: no-repeat; }
opacity
Sets transparency level from 0 (invisible) to 1 (fully visible).
.fade { opacity: 0.5; }
.hidden { opacity: 0; }
.visible { opacity: 1; }
Box Model (Spacing)
width
Sets the width of an element.
.container { width: 800px; }
.box { width: 50%; }
img { width: 100%; }
height
Sets the height of an element.
.banner { height: 400px; }
.box { height: 100vh; }
padding
Space inside an element, between content and border. Can set all sides or individually.
.box { padding: 20px; }
.card { padding: 10px 20px; }
.container { padding-top: 30px; }
margin
Space outside an element, between it and other elements.
h1 { margin: 20px; }
p { margin: 10px 0; }
.centered { margin: 0 auto; }
border
Creates a border around an element. Specify width, style, and color.
.box { border: 2px solid black; }
.card { border: 1px dashed #ccc; }
img { border: 3px solid red; }
border-radius
Rounds the corners of an element.
.button { border-radius: 5px; }
.circle { border-radius: 50%; }
.card { border-radius: 10px 10px 0 0; }
box-shadow
Adds a shadow effect around an element.
.card { box-shadow: 0 4px 6px rgba(0,0,0,0.1); }
.button { box-shadow: 2px 2px 5px gray; }
Display & Layout
display
Controls how element is displayed: block, inline, inline-block, flex, grid, none.
.hidden { display: none; }
.inline { display: inline; }
.flex-container { display: flex; }
position
Controls positioning method: static, relative, absolute, fixed, sticky.
.navbar { position: fixed; }
.overlay { position: absolute; }
.card { position: relative; }
top / right / bottom / left
Positions element when using position: absolute, relative, or fixed.
.navbar { top: 0; left: 0; }
.corner { top: 10px; right: 10px; }
float
Makes element float to left or right, allowing text to wrap around it.
img { float: left; }
.sidebar { float: right; }
clear
Controls behavior around floated elements.
.footer { clear: both; }
.section { clear: left; }
z-index
Controls stacking order of overlapping elements. Higher numbers = on top.
.navbar { z-index: 100; }
.modal { z-index: 1000; }
overflow
Controls what happens when content is too big: visible, hidden, scroll, auto.
.container { overflow: hidden; }
.scrollbox { overflow: scroll; }
.text { overflow: auto; }
Flexbox Properties
display: flex
Turns an element into a flex container, enabling flexbox layout for its children.
.container { display: flex; }
justify-content
Aligns flex items horizontally: flex-start, center, flex-end, space-between, space-around.
.container { justify-content: center; }
.nav { justify-content: space-between; }
align-items
Aligns flex items vertically: flex-start, center, flex-end, stretch.
.container { align-items: center; }
.box { align-items: flex-start; }
flex-direction
Sets direction of flex items: row, column, row-reverse, column-reverse.
.container { flex-direction: row; }
.vertical { flex-direction: column; }
gap
Sets spacing between flex or grid items.
.container { gap: 20px; }
.grid { gap: 10px 20px; }
Effects & Transitions
transition
Creates smooth transitions when properties change. Specify property, duration, and timing.
.button { transition: all 0.3s ease; }
a { transition: color 0.2s; }
transform
Applies transformations: rotate, scale, translate, skew.
.rotate { transform: rotate(45deg); }
.grow { transform: scale(1.2); }
.move { transform: translateX(50px); }
cursor
Changes mouse cursor appearance: pointer, default, grab, text, etc.
.button { cursor: pointer; }
.disabled { cursor: not-allowed; }
:hover
Pseudo-class that applies styles when user hovers over an element.
button:hover { background-color: blue; }
a:hover { color: red; }
Other Useful Properties
list-style
Controls list marker style. Can remove bullets/numbers with 'none'.
ul { list-style: none; }
ol { list-style: decimal; }
li { list-style-type: square; }
max-width / max-height
Sets maximum width or height. Useful for responsive design.
.container { max-width: 1200px; }
img { max-width: 100%; }
min-width / min-height
Sets minimum width or height.
.box { min-height: 400px; }
.sidebar { min-width: 250px; }
object-fit
Controls how images/videos fit in their container: cover, contain, fill.
img { object-fit: cover; }
.logo { object-fit: contain; }
visibility
Shows or hides element while keeping its space: visible or hidden.
.hidden { visibility: hidden; }
.visible { visibility: visible; }
Advanced Effects
backdrop-filter
Applies effects like blur to the area behind an element. Great for glass-morphism effects.
.glass { backdrop-filter: blur(10px); }
.frosted { backdrop-filter: blur(5px) brightness(1.1); }
filter
Applies visual effects to an element: blur, brightness, contrast, grayscale, etc.
img { filter: grayscale(100%); }
.bright { filter: brightness(1.5); }
.shadow { filter: drop-shadow(0 4px 8px black); }
animation
Applies keyframe animations to elements. More powerful than transitions.
@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

.fade { animation: fadeIn 1s ease; }
:active
Pseudo-class that applies styles when an element is being clicked/activated.
button:active { transform: scale(0.95); }
.link:active { color: red; }
:focus
Pseudo-class that applies styles when an element has keyboard focus.
input:focus { border-color: blue; }
button:focus { outline: 2px solid #FCA311; }
CSS Grid Properties
display: grid
Turns an element into a grid container for 2D layouts.
.container { display: grid; }
grid-template-columns
Defines the column structure of the grid.
.grid { grid-template-columns: 1fr 1fr 1fr; }
.layout { grid-template-columns: 200px auto 200px; }
.responsive { grid-template-columns: repeat(3, 1fr); }
grid-template-rows
Defines the row structure of the grid.
.grid { grid-template-rows: 100px auto 50px; }
.equal { grid-template-rows: repeat(3, 1fr); }
Responsive Design
@media
Creates responsive styles that apply only at certain screen sizes (media queries).
@media (max-width: 768px) {
  .container { flex-direction: column; }
}

@media (min-width: 1200px) {
  .content { max-width: 1400px; }
}
viewport units (vw, vh)
Units relative to viewport size: vw (viewport width), vh (viewport height).
.hero { height: 100vh; }
.full-width { width: 100vw; }
.half { width: 50vw; }
CSS Comments
/* comment */
Creates a comment in CSS that won't affect styling. Useful for notes and organization.
/* Main navigation styles */
